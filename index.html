My best Work

LibroMind

I decided for my bachelor's degree project to make a web application for libraries. The app is going to give the user the possibility to browse through the chosen library's catalog. Some of the key features are the possibility to make an order to borrow a book, to extend the period of the borrowing, and to see and give reviews to the books you want. As for the admin part, the librarians have the possibility to manage their books catalog in order to add, delete or edit a book. You can find the back-end here, and the front-end here.

Technologies used include:

  • ASP .Net Core
  • Angular
  • HTML
  • CSS
  • TypeScript
  • SQL Server
